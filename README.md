# Online_Checkers_and_chess
A full-stack multiplayer web application where users can play Checkers and Chess in real time against other players online.
Tech Stack

Backend: Python (Flask), Flask-SocketIO
Frontend: JavaScript, HTML, CSS
Database: SQLite
Real-time Communication: Socket.IO

Features

Real-time multiplayer gameplay with WebSocket connections
Interactive game boards with drag-and-drop or click-to-move mechanics
User authentication and session management
Game state persistence with SQLite
Responsive UI for desktop and mobile

Getting Started
cd online_Checkers
pip install -r requirements.txt
python app.py
